# Laravel + React 

Front-end 
- [x] Laravel 10 + PHP 8

Back-end 
- [x] React 18 + TypeScript + Vite
- [x] Tailwind CSS / SASS

### Set Up

Install dependencies 
```
composer install && npm install
``` 

Run Laravel and React dev server concurrently
```
npm run dev
``` 

Preview production build 
```
npm run production
```
